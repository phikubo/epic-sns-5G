Flujo de Simulación

  • Paso 1
    Ingresar parámetros de simulación.
  • Paso 2
    Realizar validaciones iniciales.
  • Paso 3
    Realizar pre-simulación con una muestra de distancia fija.
  • Paso 4
    Realizar simulación con valores aleatorios.
  • Paso 5
    Calcular Potencia Recibida, SINR, Throughput.
  • Paso 6
    Iterar N veces desde el Paso 4.
  • Paso 7
    Obtener estadísticas.

Características Técnicas

A continuación, una serie de caracteristicas técnicas de SAMI v1.

Criterio de Mayor Potencia.
Los usuarios reciben señal de diferentes celdas. El criterio de mayor potencia recibida se aplica para seleccionar una sola celda. Cuando el usuario recibe la misma potencia de dos celdas diferentes, SAMI selecciona aleatoriamente una celda de conexión.
Producción de Imágenes
En SAMI es posible producir una imagen de la intensidad de potencia recibida en un área determinada y con una resolución dinámica. Sin embargo, esta característica está reservada para una única simulación por la alta carga de procesamiento que se requiere para generarla.
Antenas Trisectorizadas
El algorítmo de trisectorización es dinámico y puede configurarse para generar más lóbulos ideales y de igual forma para generar diferentes patrones de radiación si se dispone de las ecuaciones matemáticas que los modelan.
Montecarlo y Proceso Puntual Poisson.
La posición de los usuarios se distribuyen en un área determinada de manera aleatoria y uniforme. Cada iteración corresponde a una versión aleatoria de un mismo escenario de simulación, de esta forma las iteraciones convergen a la estadística de análisis; este método iterativo se conoce como Montecarlo. En conjunto con este método, la cantidad de usuarios generados en cada simulación corresponde al Proceso Puntual Poisson, el cual sigue la distribución de Poisson, sin embargo, la intensidad (lambda) debe entenderse en este caso como una medida de densidad poblacional, que depende del área disponible para generar los usuarios y cuya relación es directamente proporcial: a mayor área disponible, mayor la cantidad de usuarios.
